# Snake Water Gun Game 🐍💧🔫

A simple command-line Snake Water Gun game built using Python.

## About the Game

Snake Water Gun is similar to Rock Paper Scissors:

* Snake 🐍 drinks Water 💧 → Snake wins
* Water 💧 drowns Gun 🔫 → Water wins
* Gun 🔫 kills Snake 🐍 → Gun wins

The player chooses Snake, Water, or Gun, and the computer randomly selects its choice. The winner is then determined based on the game rules.

## Features

* Random computer choice using Python's `random` module
* User input handling
* Winner determination logic
* Command-line interface
* Beginner-friendly Python project

## Technologies Used

* Python 3
* Random Module

## How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/snake-water-gun-game.git
   ```

2. Navigate to the project folder:

   ```bash
   cd snake-water-gun-game
   ```

3. Run the program:

   ```bash
   python snake_water_gun.py
   ```
